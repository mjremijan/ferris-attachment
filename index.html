About

Attachment is a simple application designed to send an email with a file as an attachment. When you start the application, use the -unsent command line argument to specify which directory contains files which have NOT been emailed yet. Similarly, use the -sent command line argument to specify where to put a file AFTER it has been emailed. Refer to the Command line arguments page for more details.

This project was originally designed and developed to work with Ferris Reading. Reading takes a large PDF document and splits it into any number of smaller documents so it's reasy to real the large PDF by breaking it into more managable smaller pieces. Question is how do you get those smaller pieces to you to read? That's what Attachment is for. Attachment is scheduled to run each morning to email one of those smaller pieces. Of course this is only one possible use for Attachment.
